# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ПРОГРАММЫ

## Быстрый старт

1. **Откройте проект в Visual Studio:**
   - Запустите Visual Studio
   - Выберите "Открыть проект или решение"
   - Выберите файл `KursovayaWork.sln`

2. **Запустите программу:**
   - Нажмите F5 или выберите "Отладка" → "Начать отладку"

## Пошаговая инструкция

### Шаг 1: Формирование данных

1. В главном меню выберите пункт **"1. Формирование исходных данных"**
2. Выберите способ ввода:
   - **"1"** — Ручной ввод данных
   - **"2"** — Генерация случайных данных

#### Ручной ввод:
- Введите минимальную требуемую калорийность (например, 2000)
- Введите количество видов продуктов (например, 4)
- Для каждого продукта введите:
  - Вес одной упаковки (кг)
  - Калорийность одной упаковки (ккал)
  - Максимальное количество упаковок

#### Генерация случайных данных:
- Введите минимальную требуемую калорийность
- Введите количество видов продуктов
- Программа автоматически сгенерирует данные

### Шаг 2: Просмотр данных

1. В главном меню выберите пункт **"2. Вывод исходных данных"**
2. На экране отобразится таблица со всеми продуктами и их характеристиками

### Шаг 3: Решение задачи

1. В главном меню выберите пункт **"3. Решение задачи"**
2. Выберите метод решения:
   - **"1"** — Эвристический метод (быстро, но может быть неоптимальным)
   - **"2"** — Полный перебор (медленно, но гарантирует оптимальность)
   - **"3"** — Оба метода (сравнение результатов)

3. Просмотрите результаты:
   - План закупки (количество каждого продукта)
   - Общий вес
   - Общая калорийность
   - Время выполнения алгоритма

### Шаг 4: Сохранение данных

1. В главном меню выберите пункт **"4. Сохранение исходных данных"**
2. Выберите формат:
   - **"1"** — Текстовый файл (.txt)
   - **"2"** — Двоичный файл (.dat)
3. Введите имя файла (расширение добавится автоматически)

### Шаг 5: Загрузка данных

1. В главном меню выберите пункт **"5. Восстановление исходных данных"**
2. Выберите формат файла:
   - **"1"** — Текстовый файл (.txt)
   - **"2"** — Двоичный файл (.dat)
3. Введите имя файла

## Пример работы

### Пример 1: Быстрое решение

1. Выберите "1" → "2" (генерация данных)
2. Введите: калорийность = 2000, количество продуктов = 5
3. Выберите "2" (просмотр данных)
4. Выберите "3" → "1" (эвристический метод)
5. Просмотрите результаты

### Пример 2: Точное решение (для малых задач)

1. Выберите "1" → "1" (ручной ввод)
2. Введите данные для 3-4 продуктов
3. Выберите "3" → "2" (полный перебор)
4. Просмотрите оптимальное решение

## Формат текстового файла

Текстовый файл имеет следующий формат:
```
2000.0
4
0.5;250;6
1.2;600;3
0.8;400;5
1.0;350;4
```

- Первая строка: минимальная калорийность
- Вторая строка: количество продуктов
- Далее: для каждого продукта — вес;калорийность;максимальное_количество

## Рекомендации

1. **Для больших задач** (более 8-10 продуктов) используйте эвристический метод
2. **Для малых задач** (до 6-7 продуктов) можно использовать полный перебор для получения оптимального решения
3. **Сохраняйте данные** перед экспериментированием с различными методами
4. **Проверяйте результаты** — убедитесь, что калорийность не меньше требуемой

## Устранение неполадок

**Проблема:** Программа не запускается
- **Решение:** Убедитесь, что установлен .NET 6.0 SDK или выше

**Проблема:** Ошибка при загрузке файла
- **Решение:** Проверьте формат файла и путь к файлу

**Проблема:** Полный перебор работает очень долго
- **Решение:** Это нормально для больших задач. Используйте эвристический метод или уменьшите количество продуктов/максимальные количества
