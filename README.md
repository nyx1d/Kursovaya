# Курсовая работа по дисциплине "Исследование операций"

## Вариант 6: Оптимизация закупки продуктов для похода

### Описание задачи

Турист, собираясь в поход, закупает продукты в неделимых упаковках известного веса C_j и калорийности a_j, j=1..n. Количество продуктов каждого вида можно купить не более d_j упаковок, j=1..n.

**Требуется:** определить план закупки продуктов, чтобы их суммарная калорийность была не ниже К килокалорий, а общий вес был минимальным.

### Требования

- .NET 6.0 SDK или выше
- Visual Studio 2019/2022 или Visual Studio Code (опционально)

### Запуск программы

#### Способ 1: Через Visual Studio

1. Откройте файл `KursovayaWork.sln` в Visual Studio
2. Нажмите F5 или выберите "Запустить" для запуска программы

#### Способ 2: Через командную строку

```bash
cd KursovayaWork
dotnet run
```

### Функциональность программы

Программа предоставляет следующее меню:

1. **Формирование исходных данных**
   - Ручной ввод данных
   - Генерация случайных данных

2. **Вывод исходных данных**
   - Табличное представление всех продуктов с их характеристиками

3. **Решение задачи**
   - Эвристический метод (жадный алгоритм)
   - Полный перебор (точное решение)
   - Сравнение обоих методов

4. **Сохранение исходных данных**
   - В текстовый файл (.txt)
   - В двоичный файл (.dat)

5. **Восстановление исходных данных**
   - Из текстового файла
   - Из двоичного файла

### Формат файлов

#### Текстовый файл (.txt)
```
2000.0
4
0.5;250;6
1.2;600;3
0.8;400;5
1.0;350;4
```
Первая строка - минимальная калорийность, вторая - количество продуктов, далее - данные о каждом продукте (вес;калорийность;максимальное_количество).

#### Двоичный файл (.dat)
Типизированный двоичный файл с той же структурой данных.

### Пример использования

1. Запустите программу
2. Выберите пункт меню "1" для формирования данных
3. Выберите "1" для ручного ввода или "2" для генерации
4. Введите данные
5. Выберите пункт меню "2" для просмотра данных
6. Выберите пункт меню "3" для решения задачи
7. Выберите метод решения (1, 2 или 3 для сравнения)
8. Просмотрите результаты
